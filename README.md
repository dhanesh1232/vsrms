# VSRMS — Care & Move Platform

A unified healthcare and mobility management system designed to connect medical care services with specialized patient transport solutions.

## 🚀 Phase 1 Overview
Phase 1 focuses on establishing the digital presence and enquiry management system for the Care and Move verticals.

### Core Modules
- **Unified Public Website**: High-performance, SEO-optimized landing pages.
- **Enquiry Management**: Automated capture and admin dashboard for patient/transport requests.
- **Inbuilt Backend**: Next.js 15+ based architecture with MongoDB integration.
- **Premium Design**: Modern, responsive UI with glassmorphism and smooth animations.

## 🛠 Tech Stack
- **Frontend**: Next.js 15 (App Router), React 19, TypeScript.
- **Styling**: Tailwind CSS 4, Framer Motion, Lucide Icons.
- **Backend**: Next.js Server Actions & API Routes, Mongoose.
- **Tools**: pnpm, Biome, Zod.

## 📂 Project Structure
```text
src/
├── app/          # Next.js App Router (Routes & API)
├── components/   # Reusable UI & Layout components
├── lib/          # Utilities & DB connections
├── models/       # Mongoose Schemas (Enquiry, User)
├── styles/       # Global CSS & Design Tokens
doc/              # Architectural & Feature Documentation
```

## ⏱ Installation & Setup
1. Clone the repository.
2. Install dependencies: `pnpm install`.
3. Configure `.env.local` using `.env.example`.
4. Run development server: `pnpm dev`.
